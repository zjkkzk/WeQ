# 40800 — 消息正文（MsgBody）

`40800` 是 `c2c_msg_table` / `group_msg_table` / `dataline_msg_table` 中存贮消息正文的列，为 protobuf，结构为可重复的消息段（`ElementWire`）序列——类似富文本，一条消息可包含多个消息段，按内容顺序排列，部分类型可嵌套。
