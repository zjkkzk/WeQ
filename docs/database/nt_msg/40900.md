# 40900 — 消息缓存（MsgCache）

`40900` 存贮转发 / 引用场景下缓存的源消息快照，为 protobuf，结构为可重复的 `MsgCache` 记录，且可递归嵌套（转发中套转发）。

以列 `40011`（msgType）区分用途：

| 40011 值 | 含义     | 40900 内容                 |
| -------- | -------- | -------------------------- |
| 8        | 合并转发 | 被转发的源消息缓存         |
| 9        | 回复引用 | 被引用的源消息缓存         |
